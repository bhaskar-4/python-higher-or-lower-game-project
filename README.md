# python-higher-or-lower-game-project
A simple command-line based Higher or Lower game built using Python. The game challenges players to guess which of two options has a higher value (such as followers, score, or any numerical metric), testing their intuition and decision-making skills.

🚀 Features<br>
🎲 Randomized game data every round<br>
🔁 Continuous gameplay loop<br>
📈 Score tracking system<br>
🧩 Modular and readable code<br>
⚡ Fast and lightweight CLI-based execution<br><br>
🕹️ How It Works<br>
Two options (A and B) are displayed,
Each option has a hidden numerical value
Player guesses which one is higher<br>
✅ Correct → Score increases, game continues<br>
❌ Wrong → Game ends, final score shown<br><br>
🛠️ Tech Stack<br>
Language: Python<br>
Libraries Used:<br>
random (for generating dynamic comparisons)<br>
